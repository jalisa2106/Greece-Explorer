# 🇬🇷 Greece Explorer – Travel Destination Guide

Greece Explorer is a visually engaging travel destination guide website designed to help users explore the beauty, culture, and history of Greece. The project presents important tourist destinations, historical landmarks, and travel insights through an intuitive and interactive web interface.

The website is structured as a multi-page travel guide containing **11 dedicated pages**, each focusing on different aspects of Greece such as famous cities, attractions, culture, and travel experiences. It aims to combine informative content with modern web design to create an immersive browsing experience.

## ✨ Features

* **🌍 Comprehensive Destination Guide:** Explore major Greek destinations including iconic cities, islands, and cultural landmarks.
* **🧭 11 Structured Pages:** The website is organized into multiple sections that make navigation simple and user-friendly.
* **🎨 Clean and Modern UI:** Designed with a visually appealing layout to enhance readability and engagement.
* **📱 Responsive Design:** Optimized for desktops, tablets, and mobile devices.
* **🗺️ Tourism-Focused Content:** Highlights famous attractions, historical sites, and travel experiences in Greece.
* **⚡ Interactive Navigation:** Smooth navigation between sections to create a seamless user experience.

## 🛠️ Technologies Used

* **HTML5** – Structure of the web pages
* **CSS3** – Styling and layout design
* **JavaScript** – Interactive features and dynamic behavior

## 🎯 Purpose of the Project

The primary goal of this project is to demonstrate front-end web development skills by building a structured travel guide website. It focuses on:
* Designing an organized multi-page website architecture
* Implementing responsive layouts from scratch
* Presenting travel information in an engaging format
* Practicing core UI/UX design principles
